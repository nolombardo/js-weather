# Weather App

A responsive weather application that fetches real-time weather data using the Open-Meteo API.

## Features

* Search weather by city name
* Displays temperature, wind, and conditions
* Uses weather codes mapped to icons
* Responsive layout for various screen sizes

## Built With

* HTML
* CSS
* JavaScript
* Open-Meteo API

## Live Site

[nolombardo.github.io/js-weather](https://nolombardo.github.io/js-weather/)

## Purpose

This project demonstrates API integration, asynchronous JavaScript, and responsive UI design.
